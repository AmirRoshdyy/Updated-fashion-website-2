# StyleHub - Clothing E-commerce Website

A modern, responsive clothing store built with HTML, CSS, and JavaScript. Tailored for the Egyptian market.

## Features
- 🛍️ Product catalog with 10 fashion items
- 📏 Size selection
- 🛒 Shopping cart (saved in localStorage)
- 🔐 User Authentication (Login/Register with strong password validation)
- 💳 Advanced Checkout Form (Address, +20 Phone format, Governorate selection)
- 💵 Multiple Payment Methods (Cash on Delivery & Credit Card with validation)
- 📱 Fully responsive design
- ⚡ Fast and lightweight (no frameworks)

## How to Run
1. Clone this repo: `git clone https://github.com/YOUR-USERNAME/my-clothing-store.git`
2. Open `index.html` in your browser
3. That's it!
